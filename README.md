# Social Media Application
A minimal social media platform

## Tech Stack
### Frontend

- Next.js (React, TypeScript)

- Client-side rendering for MVP

- API-based communication with backend

### Backend

- Node.js (Express-style REST APIs)

- JWT-based authentication

### Database

- MongoDB Atlas

- Document-based schema optimized for social data

### Infrastructure

- Docker (frontend + backend containers + nginx container)

- Docker Compose for local & server orchestration

- AWS EC2 for hosting

- GitHub Actions for CI/CD
